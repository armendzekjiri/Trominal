Install · v1.4.2

Pick your platform.

Server first, client second. Both take about two minutes — most of which is waiting for Docker to pull.

1. Install the server

Run the server somewhere your team can reach — a VPS, a homelab box, your laptop, doesn't matter. It needs port 8443 and a writable volume.

~/infra
# Quickstart with Docker
$ docker run -d --name trominal \
    -p 8443:8443 \
    -v trominal-data:/var/trominal \
    ghcr.io/trominal/server:1.4.2
→ trominal v1.4.2 listening on :8443
→ initial admin password: acme-foxtrot-21
 
# Or with docker compose · helm · nix · systemd
→ see docs.trominal.dev/server

3. Pair the client with your server

Open the app, paste your server URL, log in with the initial admin password (printed by the server on first run). The vault is created on-device and the master password never touches the wire.

Need something different?

Source builds, Nix flakes, ARM Linux, Raspberry Pi — it's all there. Browse the docs or ask in Discord.
