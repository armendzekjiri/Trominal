Report a bug

Improve the docs

Typo, broken example, missing platform note? Docs live in /docs. Edit on GitHub and PR — no local setup needed.

Write code

Server is Go. Desktop is Tauri + TypeScript. iOS is Swift. Pick your stack, run the dev loop, ship a PR. No CLA — just sign the DCO.

Dev setup

From clone to running app, in three commands.

Fully reproducible — every dep is pinned in flake.nix and .tool-versions.

trominal/server
# Server (Go 1.22)
$ git clone github.com/trominal/trominal
$ cd trominal/server
$ make dev
→ trominal-server :8443 · hot reload on
trominal/desktop
# Desktop (Tauri + Vite)
$ cd ../desktop
$ pnpm install
$ pnpm tauri dev
→ vite :5173 · tauri window opens

We follow the Contributor Covenant 2.1. Bigotry, harassment, and bad-faith engagement get you banned. Disagreement and strong opinions are fine and welcome.

Sign-off (DCO)

No CLA. We use the Developer Certificate of Origin — sign your commits with git commit -s to confirm you have the right to contribute the code.
